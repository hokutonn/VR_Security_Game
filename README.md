【VRセキュリティ教育ツール（攻撃者視点） 引き継ぎ資料】
作成日：2026年2月18日
作成者：重田 歩久翔 (N22026)
所属：大阪工業大学 情報セキュリティ研究室

■ 1. プロジェクト概要
攻撃者視点（ショルダーハック、クイッシング、物理的盗難）を通じて、
セキュリティ意識（当事者意識）を向上させるVR教育ツールです。
Meta Quest 3s 向けに開発されています。

※重要※ プロジェクトデータについて
本プロジェクトの完全なデータ（zipファイル）は、GitHubリポジトリの「Releases」ページにアップロードされています。
そこから最新のzipファイルをダウンロードして解凍してください。
（ファイルサイズが大きいため分割されている場合は、全てダウンロードしてから結合・解凍してください）

■ 2. 開発環境（※必ず合わせること）
このプロジェクトを開く際は、以下のバージョンを使用してください。
・Unity Version： Unity 6 (6000.0.27f1)
  ※バージョンが異なるとXR Interaction Toolkitが動作しない可能性があります。
・PC OS： Windows 11
・ターゲットデバイス： Meta Quest 3s

■ 3. 導入・セットアップ手順
1. Unity Hub を起動し、「リストに追加」から解凍したフォルダを選択。
2. 初回起動時、Libraryフォルダの再構築に時間がかかります（10分〜数十分）。
3. File > Build Settings を開き、Platform が「Android」になっているか確認。
   なっていない場合は「Switch Platform」を押す。
4. Edit > Project Settings > XR Plug-in Management で「OpenXR」または「Oculus」にチェックが入っているか確認。

■ 4. シーン構成 (Assets/Scenes/GameScenes/)
開発やビルド時は以下の順序で遷移します。
1. TitleScene.unity      : タイトル画面
2. PrologueScene.unity   : プロローグ（導入）
3. TutorialScene.unity   : 操作説明（チュートリアル）
4. GameScene.unity       : メインゲーム（オフィス環境での本編）
5. EpilogueScene.unity   : エピローグ（リザルト・結末）

■ 5. 使用している主要SDK・アセット
・XR Interaction Toolkit (VR操作の根幹)
・TextMeshPro (Assets/Fonts内に日本語フォントアセットあり)
・Pluginsフォルダ (各種プレハブ)
・Charactersフォルダ (敵NPCの設定、アニメーション等)
・Mixamo (NPCのアニメーション・モーション素材として使用)

■ 6. 既知のバグ・開発時の注意点（※重要※）
以下の挙動には注意してください。

【BossAI（上司NPC）の挙動について】
・フェーズ2以降、上司が着席する際、机にギリギリまで近づいてから
  椅子へ「テレポート（強制移動）」して座る仕様になっています。
・NavMesh Surface（移動可能範囲）を安易に変更・再ベイクすると、
  上司が机に対して変な方向を向いて座ってしまう恐れがあります。
  NavMeshをいじる際は、着席位置の座標がずれないよう注意してください。

【アイテムボックス（インベントリ）について】
・リスタート時、アイテムボックスがプレイヤーの体に近すぎると、
  ボックス内のコライダーの上にプレイヤーが乗ってしまい、挙動がおかしくなることがあります。
・修正する場合：XR Origin > Camera Offset > WaistInventory の
  「HolsterVisual」の設定をInspectorから調整し、体から少し離してください。

【タイトル画面の操作】
・現在、タイトル画面でのUI操作（ボタン押しなど）が非常に反応しにくくなっています。
・コントローラーから出る「Ray（光線）」の判定設定がうまくいっていないためです。
・もし修正できるなら、Ray Interactorの設定を見直してください（頑張って！）。

■ 7. 開発・デバッグ環境のアドバイス
VR開発を効率化するために、以下のツール・機材の使用を強く推奨します。

【Meta Quest Developer Hub (MQDH)】
・必須ツールです。PC画面のミラーリング、apkのインストール、ログ確認が容易になります。
・Meta公式サイトからダウンロードして設定してください。

【PC接続ケーブル】
・開発時は「Anker PowerLine USB-C & USB 3.0ケーブル (1.8m以上)」を使用しました。
・Meta Quest Link (Air Link含む) でPCと接続しながらUnityのエディタ上で動作確認（Play Mode）ができるため、
  いちいちビルドする手間が省け、開発効率が劇的に上がります。
・高速データ通信対応のケーブルを選んでください。
